프로젝트 환경
--
1. FrontEnd :

        React
        Chakra (리액트 UI 및 편의기능 추가 라이브러리)
        Recoil (전역상태 관리 라이브러리)


2. BackEnd : java / spring


3. DB : Mysql -> MariaDB
   (Docker 사용)


4. DBConnection : Spring Data JPA / querydsl


5. Docs : Spring REST Docs (추후 구현)
